# Research Idea

## Title
Explainable Real-Time Fraud Detection Using Machine Learning

## Problem
Credit card fraud detection systems often suffer from:
- class imbalance
- false positives
- lack of explainability

## Objective
Build an explainable real-time fraud detection system using machine learning and compare multiple models.

## Models
- Logistic Regression
- Random Forest
- XGBoost

## Dataset
Kaggle Credit Card Fraud Dataset

## Metrics
- Precision
- Recall
- F1-score
- ROC-AUC
